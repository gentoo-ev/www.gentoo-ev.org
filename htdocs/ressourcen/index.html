gentoo-ev

Diskussionen rund um das Vereinsleben betreffende Dinge

  • Abonnieren der Liste: Leere Nachricht an gentoo-ev+subscribe@lists.gentoo.org senden.
  • Nachricht an die Liste: Mail an gentoo-ev@lists.gentoo.org (wird nur von Abonnenten akzeptiert).
  • Ausführliche Anleitung (auf Englisch)

Rsync-Mirror

Zur schnelleren Verfügbarkeit werden die Gentoo-Pakete (Ebuilds) durch den Gentoo e.V. auf einem Server in Deutschland gespiegelt. Der Server kann in den Regionen rsync.europe.gentoo.org oder rsync.de.gentoo.org angesprochen werden.

Eine weiterführende Erklärung findet sich auf der Gentoo-Hauptseite: https://www.gentoo.org/support/rsync-mirrors/
